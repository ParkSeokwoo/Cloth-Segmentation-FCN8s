# Cloth-Segmentation-FCN8s
FCN8s로 cloth segmentation을 수행해보았습니다.

* 데이터 셋은 https://www.kaggle.com/datasets/rajkumarl/people-clothing-segmentation/data 여기서 다운받아 사용하였습니다.
* 코드의 핵심 부분은 깃허브의 ptsemseg에 올라와있는 코드를 참고하였습니다.
